list elements
------------------
html gives three types of list elements

1. unordered list --> <ul><li></li></ul>
    eg: 
    <ul>
        <li>milk</li>
        <li>bread</li>
        <li>eggs</li>
    </ul>

2. ordered list

    eg:
    <ol>
        <li>milk</li>
        <li>bread</li>
        <li>eggs</li>
    </ol>
3. description list

    eg:
    <dl>
        <dt>Ice Cream</dt>
        <dd>-- A soft sweet frozen food</dd>
        <dt>Tea </dt>
        <dd>-- A hot Dirnk</dd>
    </dl>
